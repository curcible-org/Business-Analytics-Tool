FAQ

Common questions.

Everything you need to know before purchasing.

Which meeting platforms does MeetCore support?

Zoom, Google Meet, and Microsoft Teams — all three in a single workflow. Each platform runs as an independent branch on the same hourly trigger. You can enable one, two, or all three depending on what your team uses. Zoom uses the Cloud Recordings API, Meet uses Google Drive transcript files, and Teams uses the Microsoft Graph API (OneDrive).

What does Teams setup require?

An Azure App Registration with Files.Read and OnlineMeetings.Read delegated permissions, and transcription enabled in your Teams Admin Center. Teams saves transcripts as VTT files to the meeting organizer's OneDrive — MeetCore polls for new files every hour. Full setup instructions are in the README.

How accurate is the action item extraction?

GPT-4o extracts action items with owner, due date, and priority from the transcript context. Accuracy is high for well-structured meetings where action items are stated clearly. The structured prompt format and examples are in the README — you can tune the extraction criteria.

Can I disable outputs I don't need?

Yes. Each output destination (Notion, Slack, Gmail, HubSpot, Google Calendar, Airtable) is a separate node. Disable any you don't use without affecting the others. Start with the ones you need most and add the rest later.

What does it cost to run?
