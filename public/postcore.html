FAQ

Common questions.

Everything you need to know before purchasing.

Which platforms does PostCore support?

Twitter/X (API v2), LinkedIn (UGC API), Instagram Business (Graph API), and Facebook Pages (Graph API). Each platform requires its own API credentials — setup guides for all four are included in the README.

Can I post to only some platforms, not all four?

Yes. Each platform is a separate node in the workflow. Disable any node and it won't fire on that run. You can start with two platforms and add the others later without rebuilding anything.

How do I control the brand voice and content style?

Via the GPT-4o prompt, configured once during setup. You define your brand voice, content pillars, and audience. Three pre-written brand voice examples are in the README. You can also supply example posts for the model to match.

Can I review posts before they go live?

The current workflow publishes directly. If you need an approval step, you can add a Slack interactive message node (documented in the README as an optional add-on) that holds publication until you approve.

What does it cost to run?

Approximately $0.05–0.10 per run for GPT-4o post generation + $0.04 per image (DALL-E 3). At 3 runs/week, running cost is roughly $5–7/month — paid directly to OpenAI.
