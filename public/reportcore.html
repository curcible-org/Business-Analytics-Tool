FAQ

Common questions.

Everything you need to know before purchasing.

What data sources does ReportCore support?

GA4 (sessions, users, bounce rate, conversions), Stripe (charges, revenue), and HubSpot (open deals, closed-won, pipeline value) out of the box. The README includes guides for adding Shopify, Mixpanel, and Airtable as additional sources.

Can I run it on-demand rather than just on Mondays?

Yes. ReportCore includes a Manual Trigger node alongside the schedule trigger. Run it any time for investor updates, board prep, or client reports. The schedule can also be changed to any cron expression — daily, bi-weekly, monthly.

How does the GPT-4o narrative work?

The aggregated metrics are passed to GPT-4o with a structured prompt that requests four specific sections: highlights, concerns, insights, and recommendations. Three prompt variants are included — growth-stage startup, established business, and agency. All are fully editable.

Does it work if I don't use one of the three data sources?

Yes. Each data source is a separate node. Disable any node you don't use and the workflow runs cleanly on the remaining sources. You can start with just Stripe and add GA4/HubSpot later.

What does it cost to run?

Approximately $0.10–0.15 per report for GPT-4o narrative generation. All data source API calls (GA4, Stripe, HubSpot) are free within normal usage limits. At 4 reports/month, running cost is under $0.60/month.
