FAQ

Common questions.

Everything you need to know before purchasing.

How do I load my knowledge base?

KnowCore includes a Python ingestion script that reads your documents (PDFs, Word files, plain text, FAQ spreadsheets), chunks them, and uploads embeddings to Pinecone. Setup takes 30–60 minutes depending on the size of your knowledge base. A step-by-step guide is included.

Will it make up answers if it doesn't know something?

No. KnowCore uses a strict RAG prompt — GPT-4o is instructed to answer only from the retrieved knowledge base context. If no relevant content is found, it responds with a fallback message and escalates to your team via Slack. Accuracy over confidence.

Does it require WhatsApp Business API access?

Yes. KnowCore uses Twilio's WhatsApp Business integration, which requires a Twilio account and WhatsApp Business API approval. The README includes the full approval and setup process — it typically takes 1–3 days for WhatsApp Business verification.

Can I use a different vector database instead of Pinecone?

Yes. The README includes adaptation guides for Supabase pgvector, Weaviate, and Qdrant. The vector search node is modular — swap the HTTP request target without changing any other part of the workflow.

Why is KnowCore priced at $99 instead of $79?

KnowCore is the most complex workflow in the suite — it includes a RAG knowledge base, dual-model pipeline (GPT-4o-mini + GPT-4o), Twilio WhatsApp integration, and a Python KB ingestion script. Setup is 4–6 hours versus 1–2 hours for simpler workflows.
