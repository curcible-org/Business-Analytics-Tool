Product Details
What each product delivers to clients
P07 MeetCore™ — AI Meeting Summary & Follow-Up All Tiers
Client receives
  • Branded email summary within 1 hour of meeting end
  • Action items with owner + deadline extracted automatically
  • Key decisions and topics listed
  • Follow-up calendar event auto-created when needed
Platforms supported
  • Zoom — OAuth2, cloud recordings, VTT transcripts
  • Google Meet — Drive API, auto-saved .vtt files
  • Microsoft Teams — Graph API, OneDrive (Growth+)
  • No-recording status email every hour
Client setup required
  • Zoom OAuth2 credentials (or Meet/Teams)
  • Google account for email + calendar delivery
  • Cloud recording enabled in Zoom settings
  • ~10 min onboarding form
Credentials Zoom OAuth2 Google OAuth2 OpenAI API Microsoft OAuth2 (Teams)
P01 InboxCore™ — AI Email Triage & Auto-Reply All Tiers
Client receives
  • Emails classified by urgency and intent automatically
  • Draft replies generated for routine inquiries
  • High-priority emails flagged for human review
  • Daily inbox digest with summary of activity
How it works
  • Polls Gmail every 15 min via OAuth2
  • GPT-4o classifies: urgent / routine / newsletter / spam
  • RAG against client's FAQ/knowledge base (Qdrant)
  • Drafts saved in Gmail for human approval gate
Client setup required
  • Gmail OAuth2 access
  • Upload knowledge base documents (FAQs, policies)
  • Define reply tone and escalation rules
  • ~20 min onboarding + doc upload
Credentials Google OAuth2 OpenAI API Qdrant API
P03 LeadCore™ — AI Lead Qualification & CRM Push Growth +
Client receives
  • Leads scored 1–10 automatically on intake
  • Qualified leads pushed to CRM with enriched profile
  • Instant personalized follow-up email sent
  • Unqualified leads filtered before touching sales team
How it works
  • Webhook trigger from form (Typeform, Webflow, etc.)
  • GPT-4o scores + qualifies against ICP criteria
  • HubSpot / Airtable CRM push via API
  • Personalized email sent via Gmail within 2 min
Client setup required
  • Form webhook URL from client's intake form
  • CRM credentials (HubSpot API or Airtable token)
  • Define ICP (ideal customer profile) criteria
  • ~30 min setup + ICP definition session
Credentials Google OAuth2 OpenAI API HubSpot API Airtable Token
P04 PostCore™ — Social Media Auto-Publisher Growth +
Client receives
  • Captions generated per platform from a content brief
  • Posts scheduled and published automatically
  • Covers LinkedIn, Instagram, X, TikTok, Facebook
  • Airtable content calendar as the control panel
How it works
  • Airtable content calendar triggers on "Approved" status
  • GPT-4o rewrites caption per platform tone/limits
  • Blotato node publishes to all connected socials
  • Status updated to "Posted" on success
Client setup required
  • Blotato account with social platforms connected

  • ~45 min setup + first content batch
Credentials Blotato API Airtable Token Google OAuth2 OpenAI API

Client Onboarding
How a client goes from signup to live in under 15 minutes
The onboarding UI is a single branded page. Client selects their tier, enters credentials for their chosen products, and you activate their workflows in n8n. No code on their end.
01
Stripe checkout
Client pays setup fee + first month via Stripe. Webhook fires to trigger onboarding.
02
Onboarding form
Branded page collects company name, email, and credentials for selected products only.
03
Credential storage
Credentials stored securely in n8n's credential vault, isolated per client workspace.
04
Workflow activation
You activate the relevant workflows in n8n and assign the client's credentials. Takes 5 min.
05
Confirmation email
Client receives a "You're live" email with what's running, what to expect, and how to reach support.
